# Users Page

After pulling the code:
To run the code:

` npm install `  

User json-server to create api locally:       
Step 1: Go to Components folder: `cd .\src\Components\`    
Step 2: `json-server --watch MOCK_DATA.json --port 4000`      

Open another terminal and enter command: 
` npm start `
